AI powers part of the host product but the user may not know it's there. No badge, no disclaimer, no sparkle icon. Results just seem better.
AI is visibly marked — badges, "AI-powered" labels, sparkle icons. No discrete CTA. Woven into the product, and the user knows it's there.
AI surfaces a specific suggestion in context — inside a host product the user is already in. Uninvoked.
An optional AI capability inside a host product. The user elects to use it — never on by default. It has its own UI surface: a button, a sidebar, a panel.
The entire product is the AI. No host product. AI is the interface, the value, and the interaction model. The user's relationship is with the AI itself.
